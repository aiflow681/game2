【游戏名称】过马路小游戏

【游戏介绍】
这是一款H5原生开发的3D策略类小游戏。游戏采用Three.js引擎开发，提供立体化的游戏体验。玩家控制一个白色小方块角色，需要从起点穿过由汽车、卡车和森林等组成的复杂道路，最终到达对面的终点。游戏具有三维视角，玩家需要观察道路上的车辆行驶情况，找准时机穿越不同的车道。

【玩法说明】
操作：
- 使用屏幕上的虚拟方向按钮控制小方块移动
- 点击"↑"按钮向前移动
- 点击"↓"按钮向后移动
- 点击"←"按钮向左移动
- 点击"→"按钮向右移动
- 也可以使用键盘的方向键进行控制
- 观察道路上的车辆移动模式，找准时机穿越

规则：
- 小方块需要避开所有车辆（汽车和卡车）
- 游戏中有三种不同类型的道路：草地、森林和公路
- 森林中有树木，不能穿过树木占据的位置
- 汽车和卡车会在公路上持续移动，从不同方向穿过
- 一旦小方块碰到车辆，游戏结束
- 每成功穿越一条车道，计数器会增加1分

目标：
- 从起点（后方草地）出发，穿过所有道路类型
- 成功到达对面（终点草地）
- 在移动过程中避免碰撞
- 尽可能获得更高的分数（穿越的车道数）

Tips：
- 仔细观察车辆的移动速度和方向规律
- 在森林中移动时注意树木的位置
- 提前规划路线，不要盲目移动
- 善用"后退"功能调整位置
- 某些车辆速度较快，需要耐心等待时机

技术介绍：
- 使用HTML5 Canvas和JavaScript开发
- 响应式设计，完美适配移动端和桌面端
- 虚拟控制器专为移动设备优化，提供直观的触摸操作
- 物理引擎模拟车辆真实运动轨迹
- 动态渲染系统，根据视角自动调整画面细节
